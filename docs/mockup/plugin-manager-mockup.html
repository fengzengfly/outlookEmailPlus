设计要点
  • 插件 provider 与内置 provider 在同一个下拉列表中，不做分组
  • 插件可标注「插件」badge，但不强制（用户也可选择隐藏）
  • 选择行为完全一致：选中后切换 provider，邮箱列表和操作不变化
  • 如果插件加载失败，该选项不出现（自动降级）
