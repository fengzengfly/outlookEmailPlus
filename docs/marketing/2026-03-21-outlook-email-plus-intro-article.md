# OutlookMail Plus

OutlookMail Plus is a registration-focused mailbox manager for individuals and teams.

Unlike general-purpose email clients, it is built specifically for **registration and verification** workflows, with deep optimization around getting signup flows done faster.

Created by a Chinese university student, the project picked up more than 500 GitHub stars in just two days.

## Why OutlookMail Plus

- **Built for registration workflows**: it removes as many unnecessary steps as possible. You can copy a mailbox address with one click. After sending a verification email on a signup page, just return to the manager, click "Verification Code", and it will automatically fetch the latest message and quickly extract the code or verification link with regex.
- **Lighter and more focused**: non-core features such as sending mail are intentionally left out, so the interface stays cleaner and every part of the product is centered on getting registrations completed smoothly.
- **Broader import compatibility**: it supports mainstream mailbox providers such as Gmail, QQ, and 163, as well as custom IMAP servers. Self-hosted mailboxes are supported too. Built-in temp mailboxes also help reduce privacy exposure.
- **Automation-friendly**: it provides external APIs for batch registration workflows, including mailbox code retrieval and mailbox release.
- **Third-party notifications**: third-party notification channels are supported. Telegram is already integrated, and important mailboxes can automatically push alerts when new mail arrives.

In short, OutlookMail Plus is a mailbox manager built for registration workflows.
